# yunlong-tools

这是一个仓库用于存放一些个人编写觉得好用的工具代码，如果有兴趣可以留个star

一、show_gpu.py
一个用于连接多个算力服务器，用于显示nvidia显卡使用情况的工具。代码使用长连接，按host序列依次输出：

| 主机   | GPU索引 | GPU名称     | 总显存 (MiB) | 已用显存 (MiB) | 利用率 (%) |
|--------|---------|-------------|---------------|----------------|------------|
| 110.56.23.56:22   | 0       | GRID A100D-80C      | 81920         | 22000          | 48%        |
